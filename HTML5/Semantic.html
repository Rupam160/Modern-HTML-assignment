What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It describes the structure of a webpage using a series of elements and tags.

What is Semantic HTML?

Semantic HTML uses HTML5 elements to provide meaning and structure to the web content. These elements clearly describe their purpose and content, making the webpage more understandable for both browsers and developers.

Commonly Used Semantic Tags

  • <header>: Represents introductory content or a set of navigational links.
  • <nav>: Defines a set of navigation links.
  • <main>: Represents the main content of the document.
  • <article>: Represents a self-contained piece of content.
  • <footer>: Represents the footer of a document or section.
