Differences between HTML and HTML5

HTML HTML5
HTML stands for HyperText Markup Language and is the standard markup language for creating web pages. HTML5 is the latest version of HTML, designed to support multimedia elements like video and audio, and provide better support for web applications.
HTML does not have native support for video and audio embedding; third-party plugins like Flash were used. HTML5 provides native support for embedding video and audio using the <video> and <audio> tags.
HTML uses general-purpose elements like <div> and <span> for structuring content. HTML5 introduces semantic elements like <header>, <footer>, <article>, and <section> for better content structure and readability.
